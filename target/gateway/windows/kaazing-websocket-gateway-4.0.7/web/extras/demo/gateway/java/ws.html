Java Echo Demo

The demonstration on this page shows WebSocket communication being utilized inside a Java Applet.

For more information, see Checklist: Build Java Clients

Java Client Requirements

This demo requires the Sun Java plugin version 1.7 or higher running in a certified browser and operating system. Currently, the Java client library is not certified on:
  • Mac OS X
  • Apple Safari
  • Firefox version 2.0 or earlier
  • Opera 9.64
Your environment does not appear to meet the requirements. You can still attempt to run the demo which may appear below.
