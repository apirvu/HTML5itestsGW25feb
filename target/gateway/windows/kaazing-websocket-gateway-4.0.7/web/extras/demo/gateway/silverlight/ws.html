Silverlight Echo Demo

The demonstration on this page shows WebSocket communication being utilized inside a Silverlight application.

For more information, see Checklist: Build Microsoft .NET and Silverlight Clients

Microsoft Silverlight Client Requirements

This demo requires the Microsoft Silverlight plugin version 4.0 or greater running in a certified browser and operating system. Currently, the Microsoft Silverlight client library is not certified on:
  • Google Chrome on Mac OS X
  • Safari on Windows
  • Linux
Your environment does not appear to meet the requirements. You can still attempt to run the demo, which may appear below.
